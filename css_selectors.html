CSS Selectors

CSS Selectors are all about finding either a group of elements, or a single element
There are 4 basic types of selectors:
  • The Universal Selector (*)
  • The TAG selector (div)
  • The CLASS selector (.classname)
  • And the ID selector (#identifier)
A single selector can match multiple elements

Selecting multiple elements

Selector:

.myClass

HTML:

…
<div>
  <div class="myClass">[content]</div>
  <div id="myID" class="myClass">[content]</div>
  <div class="myClass">[content]</div>
</div>
…
You can combine these basic selectors into a single selector

Combining selectors

Selector:

div#myID.myClass

HTML:

…
<div>
  <div class="myClass">[content]</div>
  <div id="myID" class="myClass">[content]</div>
  <div class="myClass">[content]</div>
</div>
…
However, be careful not to be add too much to your selector

Both these selectors will match the same element because there can only be 1 id of myId in a document

Selectors:

div#myID.myClass
#myId

HTML:

…
<div>
  <div class="myClass">[content]</div>
  <div id="myID" class="myClass">[content]</div>
  <div class="myClass">[content]</div>
</div>
…
Grouping these selectors lets you specify where your target is in the HTML tree
There are 3 main ways to group selectors

1. Separated by spaces

2 selectors separated by a space means the second selector is anywhere inside the first selector

Selector:

#myId .myClass

HTML

…
<div id="myId">
  <div class="notMyClass">
    <div class="myClass">[content]</div>
  </div>
  <div class="myClass">[content]</div>
  <div class="myClass">[content]</div>
</div>
…

2. Separated by greater-than sign (>)

2 selectors separated by a greater-than means the second selector is a direct child of the first selector

Selector:

#myId > .myClass

HTML

…
<div id="myId">
  <div class="notMyClass">
    <div class="myClass">[content]</div>
  </div>
  <div class="myClass">[content]</div>
  <div class="myClass">[content]</div>
</div>
…

3. Separated plus sign (+)

2 selectors separated by a plus sign means the second selector is sibling of the first selector and the second selector immediately follows the first.

Selector:

#myId + .myClass

HTML

…
<div>
  <div id="myId">
    <div class="myClass">[content]</div>
  </div>
  <div class="myClass">[content]</div>
  <div class="myClass">[content]</div>
</div>
…
You can also select elements based on attributes. There are 4 different ways to select based on attributes.
Attributes are denoted by square brackets.

1. The attribute exists

Selector:

a[title]

HTML

…
 <div><a href="http://www.google.com" title="Search">Google</a></div>
 <div><a href="http://www.facebook.com" title="Social">Facebook</a></div>
 <div><a href="http://www.yahoo.com">Yahoo</a></div>
…

2. The attribute value equals some term

This is set by having the attribute name followed by the attribute value, inside of the square brackets

Selector:

a[title=Search]

HTML

…
 <div><a href="http://www.google.com" title="Search">Google</a></div>
 <div><a href="http://www.facebook.com" title="Social">Facebook</a></div>
 <div><a href="http://www.yahoo.com">Yahoo</a></div>
…

3. The attribute contains some value

This means that a whitespace-separated list of values contains the specified value

Selector:

a[title~=not]

HTML

…
 <div><a href="http://www.google.com" title="This is cool">Google</a></div>
 <div><a href="http://www.facebook.com" title="This is not cool">Facebook</a></div>
 <div><a href="http://www.yahoo.com">Yahoo</a></div>
…

4. The attribute starts with the value followed by a dash (-)

The only real use of this is when you want to select a the language part of a lang attribute

Selector:

a[lang|=en]

HTML

…
 <div lang="en-gb">Hello</div>
 <div lang="en-au">G'day</div>
 <div lang="en-us">Howdy</div>
 <div lang="fr-fr">Bonjour</div>
 <div lang="fr-ca">Allo</div>
…
