CSS Selectors

CSS Selectors are all about finding either a group of elements, or a single element
There are 4 basic types of selectors:
  • The Universal Selector (*)
  • The TAG selector (div)
  • The CLASS selector (.classname)
  • And the ID selector (#identifier)
A single selector can match multiple elements

Selecting multiple elements

Selector:

.className

HTML:

…
<div>
  <div class="myClass">[content]</div>
  <div id="myID" class="myClass">[content]</div>
  <div class="myClass">[content]</div>
</div>
…
You can combine these basic selectors into a single selector

Combining selectors

Selector:

div#myID.myClass

HTML:

…
<div>
  <div class="myClass">[content]</div>
  <div id="myID" class="myClass">[content]</div>
  <div class="myClass">[content]</div>
</div>
…
However, be careful not to be add too much to your selector

Both these selectors will match the same element because there can only be 1 id of myId in a document

Selectors:

div#myID.myClass
#myId

HTML:

…
<div>
  <div class="myClass">[content]</div>
  <div id="myID" class="myClass">[content]</div>
  <div class="myClass">[content]</div>
</div>
…
